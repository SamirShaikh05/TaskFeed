create posts, interact with others, and share media.

## ✨ Features

### 🔐 Authentication

-   User Signup & Login
-   Secure password hashing using bcrypt
-   JWT-based authentication

### 📝 Post Creation

-   Create posts with:
    -   Text
    -   Images
    -   Videos (bonus)
-   Either text or media required

### 📰 Feed

-   Public feed displaying all posts
-   Username, content, media, timestamp

### ❤️ Like System

-   Toggle like/unlike
-   Stores user IDs

### 💬 Comment System

-   Add and view comments
-   Real-time updates

### 🗑️ Delete Post

-   Only post owner can delete

### 📁 Media Handling

-   Base64 upload
-   Image: max 2MB
-   Video: max 5MB

------------------------------------------------------------------------

## 🛠 Tech Stack

Frontend: - React (Vite) - Axios

Backend: - Node.js - Express - MongoDB

Deployment: - Vercel (frontend) - Render (backend) - MongoDB Atlas
